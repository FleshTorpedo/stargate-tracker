# Stargate Watch Order — installable app

337 screen stops + 28 comics, tracked on a dialing Stargate. Progress saves on-device.

## Put it on your phone (one-time, ~5 min)
1. Create a **public GitHub repo** and upload ALL files in this folder to its root.
2. Repo **Settings -> Pages -> Source: Deploy from a branch -> main / (root)** -> Save.
3. Wait ~1 min, open the Pages URL on your phone in **Chrome**.
4. Chrome menu (3 dots) -> **Add to Home screen -> Install**.

You get a real app icon, fullscreen launch, offline support, and progress saved to the phone.
Use **Backup** in the app's toolbar to export progress as a file; **Restore** to load it anywhere.

Any static host works (Netlify, Cloudflare Pages, your homelab) — HTTPS is required for install + offline.
